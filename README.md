# ML Journey

A structured, project-based learning path toward becoming a Full-Stack Robotics & AI Engineer.

## Goal

Build the skills to work on humanoid and complex robotic systems — covering ML/Deep Learning, Robotics Engineering, and Hardware/Embedded Systems.

## Roadmap

| Phase | Topics | Project |
|-------|--------|---------|
| Month 1–2 | PyTorch fundamentals, CNNs, ROS2 basics, ESP32 + IMU | Static spider robot chassis |
| Month 2–3 | YOLOv8, Inverse Kinematics, ros2_control | First walking gait |
| Month 3–4 | PID control, Kalman Filter, Transformer architecture | Adaptive gait control |
| Month 4–5 | SLAM, Nav2, Reinforcement Learning (PPO) | Autonomous mapping |
| Month 5–6 | VLA models, Behavior Cloning, 3D Computer Vision | Capstone: autonomous inspection robot |

## Stack

- **ML:** PyTorch (CPU → GPU), YOLOv8, Isaac Lab
- **Robotics:** ROS2 Humble, Nav2, MuJoCo
- **Hardware:** ESP32, Raspberry Pi 4, MPU-6050
- **OS:** Arch Linux
